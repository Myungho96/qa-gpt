# Claude Code Configuration

This file documents how Claude Code is configured for this project.

## Agent skills

### Issue tracker

Issues live in GitHub Issues. External PRs are not treated as a triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Five canonical triage roles map directly to label names. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context layout: `CONTEXT.md` + `docs/adr/` at repo root. See `docs/agents/domain.md`.
